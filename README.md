# 3dPlatform

使用Threejs开源框架，实现WEB端三维模型的定制化展示

项目源码地址：[https://github.com/TONKIA/3dPlatform](https://github.com/TONKIA/3dPlatform)

# 项目运行

```bash
cnpm install
nodemon app
```
